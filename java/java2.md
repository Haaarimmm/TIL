# Datatype and Operator
## Datatype
- int: `5`
- double: `1.1 `
- String: `"Hello"`
- charater: `'H'`

## Operator
- `+, -, *, /`
- `Math.floor / Math.ceil`: 내림, 올림
- `Math.PI`: 3.1415926535

## String
- String concatenation: `+`
- `[String].length();`
- `[String].replace("old", "new")`

# Variable Casting
- `double a = 1`: 가능
- `int b = 1.1`: 불가능
- `int b = (int) 1.1`: Casting / loss 발생
- 1 to String: `Integer.toString(1)`
- `A.getClass()`: type 반환
